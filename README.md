# Going Greek Landing Page

A premium-style static landing page for the **Going Greek** Play Store app.

## Preview locally

From the project root, run:

```bash
python3 -m http.server 4173
```

Then open:

- `http://localhost:4173/`

## Update Play Store link

In `index.html`, replace the placeholder CTA link (`href="#"`) with your real Google Play URL.
